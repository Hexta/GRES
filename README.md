GRES
====

Anisotropic wet etching simulator of silicon plates.

Support KMC and CA methods.

===

Supported platforms:
  * Windows XP-7
   - MinGW
   - MSVC 2013
   - Clang
  * GNU/Linux
   - GCC 4.7+
   - Clang

Installation

<code>cmake -DCMAKE_BUILD_TYPE=Release . && make && make install</code>
